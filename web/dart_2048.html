Règles du jeu 2048

Le but du 2048 est de regrouper les cases ayant les mêmes valeurs pour qu'elles s'additionnent. Plus vous additionnerez de grandes valeurs, plus votre pointage grandira rapidement!
Le but ultime? Parvenir à créer une case qui aura une valeur de 2048. Bonne chance!

Trois niveaux

Comme vous l'avez sûrement remarqué(e), il est possible sur cette version de changer la taille du jeu. Plus le jeu est grand, plus c'est facile d'accumuler des points! Mais... Pour compliquer un peu le jeu, vous devrez accumuler davantage pour gagner le jeu: 4096 pour le jeu 5x5 et 8192 pour le jeu 6x6. Aurez-vous la patience qu'il faut?

Quelques trucs

- La fenêtre se déplace avec les flèches du clavier? Cliquez sur le tableau de jeu et ça devrait se corriger.

- Le jeu est trop grand pour votre écran? Utilisez les touches CTRL- et CTRL+ pour ajuster la taille.
